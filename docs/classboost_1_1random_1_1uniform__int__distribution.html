詳解

template<class IntType = int>
class boost::random::uniform_int_distribution< IntType >

The class template uniform_int_distribution models a \random_distribution. On each invocation, it returns a random integer value uniformly distributed in the set of integers {min, min+1, min+2, ..., max}.

The template parameter IntType shall denote an integer-like value type.
