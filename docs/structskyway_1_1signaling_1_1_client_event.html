bool skyway::signaling::ClientEvent::ValidateJsonSize ( ) const

オーダーオブジェクトをJSONに変換したときのサイズが規定サイズ以下の場合にtrueを返します。

JS-Clientではコンストラクタでsizeバリデーションをしていますが、C++ではコンストラクタで例外を返すことはアンチパターンなので、バリデーションメソッドを実装しています。
