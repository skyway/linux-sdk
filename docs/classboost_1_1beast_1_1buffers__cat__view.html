◆ value_type

template<class... Buffers>
using boost::beast::buffers_cat_view< Buffers >::value_type = buffers_type<Buffers...>

The type of buffer returned when dereferencing an iterator. If every buffer sequence in the view is a MutableBufferSequence, then value_type will be net::mutable_buffer. Otherwise, value_type will be net::const_buffer.
